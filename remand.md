# 欧路词典剪贴板查词工具

把“复制单词 -> 切换词典 -> 输入查询”压缩成一次自然复制。

这是一个面向 Windows 用户的轻量小工具。它会在后台监听剪贴板，当你复制单个英文单词时，自动唤起欧路词典迷你查词窗口，让查词动作几乎没有打断感。

## 为什么值得用

- 更快：复制就是查词，不再手动切窗口
- 更轻：AutoHotkey v2 实现，依赖少，启动快
- 更克制：只处理单个英文单词，减少误触发
- 更实用：已经过真实联调，能拉起欧路词典迷你查词

## 适合谁

- 高频阅读英文网页、论文、PDF 的用户
- 想把查词动作做成“肌肉记忆”的学习者
- 需要一个足够小、足够直接的桌面效率工具的人

## MVP 已完成

- 监听剪贴板
- 过滤单个英文单词
- 自动调用 `eudic://lp-dict/<word>`
- 500ms 去抖
- 1500ms 同词去重
- `run.bat` 双击启动

## 已完成验收

- AutoHotkey v2 已实机运行
- 欧路词典协议已打通
- 合法单词复制后可打开“迷你查词”
- 中文和带标点文本不会误触发

## 项目亮点

- 学习成本低：代码量小，容易二次开发
- 场景清晰：只解决一个高频动作，但解决得很直接
- 适合扩展：后续可以加入托盘、热键、白名单应用、短语模式

## 演示文案

复制一个单词，词典自己弹出来。

这个项目把查词从一个“主动操作”变成一个“被动响应”的动作。你继续阅读，工具负责把单词送到欧路词典，体验上更像系统能力，而不是额外软件。
